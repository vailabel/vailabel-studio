# Getting Started

Welcome to Vision AI Label Studio! This guide will help you get up and running in just a few steps.

## Before you install

Make sure your machine meets the [System Requirements](/docs/system-requirements). The core labeling app is lightweight; the optional AI features benefit from an NVIDIA GPU.

## Quick Installation

1. Go to the [Vision AI Label Studio website](https://studio.vailabel.com/) and download the latest version for your operating system (Windows or macOS).
2. Open the downloaded file and follow the installation instructions for your platform.
   - If you see a warning about an unsigned app, see our [desktop installation guide](/docs/install-on-desktop) for troubleshooting steps.

That's it! No configuration is required for most users.

## Optional: enable AI features

Vision AI Label Studio ships with offline, on-device AI to speed up labeling. These features are optional and run locally — no cloud, no uploads.

- [AI & GPU Setup](/docs/ai-gpu-setup) — install the local AI runtime and (optionally) enable GPU acceleration.
- [AI Copilot](/docs/ai-copilot) — detect, segment, caption, and QA your images from a chat.
